1. PROJECT OVERVIEW & ARCHITECTURE
--------------------------------------------------------------------------------
The MYK Laticrete Ingestion Engine is a production-grade catalog crawler, parser, and
knowledge graph constructor designed to extract product specifications, classifications,
substrate compatibilities, and packaging options from the MYK Laticrete catalog.

Data flow:
[Seed URLs] ──> [crawl_node] ──> SQLite (crawl_records)
                      │
                      ▼
                [parse_node] ──> SQLite (products, product_versions) + QA Report
                      │
                      ▼ (Next Phase: Task 3)
                [embed_node] ──> Neo4j Graph Database & Chroma Vector Store
                      │
                      ▼
                 [FastAPI] ──> Retrieval Agent (ReAct)

3. SQLITE DATABASE SCHEMA & PERSISTENCE (data/crawl.db)
--------------------------------------------------------------------------------
All database interactions use SQLite in Write-Ahead Logging (WAL) mode to permit
concurrent reads during ingestion pipeline executions.

Three Tables:
1) crawl_records (Raw crawl dump)
   - url TEXT PRIMARY KEY
   - content_type TEXT
   - text TEXT
   - fetched_at TEXT
   - hash TEXT

2) products (Task 2 Output Table - The source of truth for Task 3 Knowledge Graph)
   - id INTEGER PRIMARY KEY AUTOINCREMENT
   - sku TEXT (nullable)
   - product_name TEXT NOT NULL (unique check target)
   - product_family TEXT
   - data_json TEXT NOT NULL (Full structured ProductRecord JSON)
   - confidence REAL (0.0 to 1.0)
   - needs_review INTEGER (1 if confidence < 0.6 or schema error, 0 otherwise)
   - version INTEGER DEFAULT 1
   - extracted_at TEXT

3) product_versions (Historical tracking)
   - id INTEGER PRIMARY KEY AUTOINCREMENT
   - sku TEXT NOT NULL (version tracking key: falls back to lowercase product name if SKU is null)
   - version INTEGER NOT NULL
   - data_json TEXT NOT NULL
   - extracted_at TEXT NOT NULL
   - diff_summary TEXT (human-readable string summarizing exactly what changed)
